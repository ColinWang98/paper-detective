## 🧪 Test Run (Day 20 Afternoon)

### Dry Run Protocol

**Purpose**: Verify entire test session workflow works end-to-end

**Who**: Facilitator + team member as pretend participant

**Steps**:

**Part 1: Pre-Test** (5 minutes)
- [ ] Open deployment URL
- [ ] Start screen/audio recording
- [ ] Complete pre-test questionnaire
- [ ] Sign consent form

**Part 2: Tasks** (20 minutes)
- [ ] Task 1: Upload PDF (time: _____)
- [ ] Task 2: Generate Brief (time: _____)
- [ ] Task 3: Advanced Search (time: _____)
- [ ] Task 4: Export Markdown (time: _____)
- [ ] Task 5: Organize Highlights (time: _____)
- [ ] Task 6: AI Clue Cards Navigation (time: _____)

**Part 3: Post-Test** (5 minutes)
- [ ] Complete SUS questionnaire (all 10 questions)
- [ ] Answer NPS question with follow-up
- [ ] Complete feature ratings

**Part 4: Interview** (10 minutes)
- [ ] Ask top 5 interview questions
- [ ] Probe for deeper insights

**Part 5: Debrief** (5 minutes)
- [ ] Thank participant
- [ ] Answer participant questions
- [ ] Stop recording
- [ ] Save recording

**Part 6: Post-Session** (immediately after)
- [ ] Review recording (check quality)
- [ ] Calculate SUS score (test calculator)
- [ ] Complete task tracking sheet
- [ ] Note any issues with materials or process
