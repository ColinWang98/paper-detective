## Time Tracking Best Practices

### During Session

**DO**:
- [ ] Start timer when participant begins task
- [ ] Stop timer when task is completed (or participant gives up)
- [ ] Note pauses and confusion separately
- [ ] Record exact times (minutes and seconds)
- [ ] Document what caused delays

**DON'T**:
- [ ] Guess times (use actual timer)
- [ ] Include facilitator instructions in task time
- [ ] Forget to record sub-task times
- [ ] Round to nearest minute (lose precision)

### After Session

**DO**:
- [ ] Compare to target times immediately
- [ ] Note significant variances (>±50%)
- [ ] Document reasons for variances
- [ ] Identify patterns across participants

**Analysis Questions**:
- Which tasks consistently take longer? Why?
- Which tasks are faster than expected? Why?
- Are there learning effects across tasks?
- Do task times correlate with SUS/NPS?
