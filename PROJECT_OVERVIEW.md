# Paper Detective - 项目总览

## 🎯 项目简介

**Paper Detective** 是一个游戏化的科研文献阅读工具，将枯燥的文献阅读变成侦探解谜体验。

> 将科研文献重构为案发现场，让用户扮演侦探收集线索、拼凑真相。

## 🎨 核心功能

### 已实现 ✅

1. **PDF阅读器**
   - 上传、翻页、缩放
   - 文本选择和高亮
   - 4色优先级标记 (HCI)

2. **侦探笔记本**
   - 两阶段工作流 (收集→整理)
   - 拖拽分组 (dnd-kit)
   - 自定义分组CRUD

3. **AI功能**
   - Clip摘要生成
   - 线索卡片 (4种类型)
   - 情报简报系统

4. **数据管理**
   - IndexedDB本地存储
   - Undo/Redo历史记录
   - 导出 (BibTeX/Markdown)
